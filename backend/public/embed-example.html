This page demonstrates how to embed an AI agent into any website using the AgentHub Embed SDK. Click the chat button in the bottom-right corner to start a conversation!

How to Use

Add the following code to your website:

<!-- 1. Include the AgentHub Embed SDK --> <script src="https://your-domain.com/embed.js"></script> <!-- 2. Initialize the chat widget --> <script> AgentHub.init({ shareToken: 'your-share-token', position: 'bottom-right', theme: 'light', width: 400, height: 600, title: 'AI Assistant', welcomeMessage: 'Hello! How can I help you today?', primaryColor: '#3b82f6', }); </script>

Advanced Usage

You can also programmatically control the chat widget:

<script> // Get the embed instance const embed = AgentHub.init({ shareToken: 'your-share-token', }); // Open the chat programmatically embed.open(); // Close the chat embed.close(); // Toggle the chat embed.toggle(); // Update configuration embed.updateConfig({ title: 'Updated Title', theme: 'dark', }); // Destroy the widget embed.destroy(); </script>

Note: Replace 'your-share-token' with your actual agent's share token and 'https://your-domain.com' with your API base URL.
